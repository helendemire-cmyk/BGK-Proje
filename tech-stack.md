# 🛠️ Teknoloji Yığını (Tech Stack) - MineCycle AI

Bu proje, kod yazımından yayına alma sürecine kadar tamamen AI destekli araçlarla geliştirilmiştir.

- **Frontend:** React.js + Tailwind CSS (Arayüz tasarımı ve kullanıcı deneyimi için).
- **AI Model:** Google Gemini 1.5 Flash (Cihaz analizi ve çevresel etki hesaplamaları için).
- **Kod Editörü:** Cursor (AI-native kodlama ve dosya yönetimi).
- **Deployment (Yayın):** Lovable / Netlify (Uygulamanın internete açılması).
- **Version Control:** GitHub (Kodların saklanması ve iş birliği).
- **API Management:** Google AI Studio (Gemini API anahtarı yönetimi).
